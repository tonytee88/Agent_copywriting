# Email Outputs

This directory contains generated email drafts in human-readable text format.

## Structure

- `*.txt` - Recent email drafts (last 7 days)
- `archive/` - Older emails organized by month

## Archival Policy

Files older than 30 days are automatically moved to `archive/YYYY-MM/` to keep the repository clean.

## File Naming

Format: `BrandName_Angle_YYYYMMDD_HHMMSS.txt`

Example: `PopBrush_Best_Deal_of_the_Year_20251209_130420.txt`
